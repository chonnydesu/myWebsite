A* Pathfinding

An a* pathfinding algorithm programmed in java that finds the shortest path between 'a' to 'b'.

*populates a random 15x15 area with random obstacles

prompts the user to input coordinates for both start/end position

*displays the optimal path based on relative cost for each possible node

Designed, Jul 2024

Forecasting Airport Delays Using Machine Learning

A project primarily coded using Jupyter Notebook that predicted future airport delays for 2025 using previous data.

used an existing database of 215,000+ samples with 20 features

visualized the data using line-plots and forecasted a prediction for the current year

uses 3 main methods, OLS, Ridge Regression and Random Forest Regression
